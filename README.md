# Inventory Sync Microservice

## 🚀 Features
- Real-time inventory updates (WebSocket)
- Low stock alerts
- Bulk updates
- Concurrency-safe (locking)
- Exception handling

## ⚡ APIs
GET /api/inventory/{storeId}
PUT /api/inventory/update
POST /api/inventory/bulk-update

## 🛠 Tech
- Spring Boot
- MySQL
- WebSocket
